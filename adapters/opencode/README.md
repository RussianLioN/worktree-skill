# OpenCode Adapter

## Purpose

Expose the portable core through an OpenCode-compatible registration surface.

## Install

1. Install `core/` into the host project.
2. Copy this adapter into the host project.
3. Follow `register-example.md` for manual or scripted registration.

## Fallback

If auto-registration is not available in the host environment, keep the adapter files installed and register the command surface manually.
